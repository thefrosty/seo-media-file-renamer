== Description ==

[Plugin homepage](http://austinpassy.com/wordpress-plugins/seo-media-file-renamer/) | [Plugin author](http://austinpassy.com/) | [Donate](https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=7431290)

This plugin enables renaming of media files for SEO purposes. It adds a field in media edit form where you can enter a new file name. After renaming, it'll regenerate all intermediate image sizes (if media file is an image), remove the old files and update all links inside posts.

== Installation ==

1. Upload `seo-media-file-renamer` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Change the file name through the 'Media' page.
